# Madhav_Store_PowerBI_Dashboard

**Project Objective:**

The owner of Madhav Store has requested our assistance in developing a dashboard to monitor and analyze their online sales performance across India.

**Dataset Description:**

The dataset provides detailed information on various categories and sub-categories related to the store's sales.

**File Type:** CSV

**Variables in the Dataset:**

**Table 1:** Orders.csv  
- Order ID
- Amount
- Profit
- Quantity
- Category
- Sub-Category
- Payment Mode

**Table 2:** Orders.csv  
- Order ID
- Order Date
- Customer Name
- City
- State

**Key Insights:**

1. Maharashtra State has earned the highest profit, with Madhya Pradesh following closely behind.
2. Harivansh is the top customer, having spent nearly 10K INR in the market.
3. The Clothing category accounts for approximately 63% of the total market.
4. Printers are the top-selling sub-category across India.
5. The majority of payments have been made using credit cards and COD (Cash on Delivery).
6. The total profit for Madhav Store across all states is 37K INR.

**Learning Outcomes:**

1. Developed an interactive dashboard to monitor and analyze online sales data.
2. Employed complex parameters for drilling down into worksheets (Orders and Details) and applied customizations using filters and slicers.
3. Established connections between tables, performed joins, and calculated fields to manipulate data, enabling user-driven parameters for visualizations.
4. Created a new column for Average Order Value (AOV) using the DAX function, providing a clear understanding of the average price per product.
5. Utilized various types of customized visualizations, including bar charts, donut charts, clustered bar charts, tiles, and slicers.
